# 🚧 vip-backend 🚧
### To start the project, just run the command:
```
yarn dev
```
## Using an Orm: TypeOrm with SQLite
[Documentation](https://typeorm.io) for more information on typeorm

## Using Express to Create a Server
```
yarn add express
```

## Using typescript to streamline development
[typesscript integration](https://docs.microsoft.com/en-us/visualstudio/javascript/compile-typescript-code-npm) with nodeJs is described in the documentation

## Migrations with typeorm
```
yarn typeorm migration:run
```
